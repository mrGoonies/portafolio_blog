# Portafolio Blog

Este es un blog personal donde comparto mis proyectos, ideas y aprendizajes en el mundo del desarrollo web y la programación con el objetivo de centralizar mis proyectos y blog en un sólo lugar.

## Contenido
- **Proyectos**: Una colección de mis proyectos más recientes y destacados.
- **Blog**: Artículos y reflexiones sobre temas de interés en tecnología y desarrollo.
- **Sobre mí**: Una breve biografía y mi trayectoria profesional.

## Stack utilizado
- **Django**: Framework web para el desarrollo del backend y agilizar el desarrollo de aplicaciones full-stack.
- **UV**: Gestor de proyectos y dependencias para Python.
- **SQLite**: Base de datos ligera para el almacenamiento de datos del blog.
